# Cat Pop: Clicker Game
## Made for Hack Club's Resolution General Coding Week 1
This project is a clicker game built with React, Vite and Tailwind CSS. There is no definite goal of the game other than to click the cat in the middle of the screen (it makes a 'pop' sound) and earn clicks to buy multipliers and auto pops.

Fun project. Love react. Tailwind is an absolute lifesaver when it comes to css.

You can visit the website [here](https://resolution-general-coding-week2.vercel.app/)
